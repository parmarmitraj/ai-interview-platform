# 🤖 AI Interview Platform
A full-stack application that uses Gemini AI to simulate technical interviews, providing real-time feedback and progress tracking.

## 🚀 Features
- **AI Question Generation:** Powered by Gemini 1.5/2.5 Flash.
- **Difficulty Levels:** Easy, Medium, and Hard modes.
- **Instant Evaluation:** Get a score (0-10) and constructive feedback.
- **Progress Dashboard:** Visualized score trends using Recharts.
- **Model Answers:** Learn from AI-generated perfect responses.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Axios, Recharts, Lucide-React
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **AI:** Google Gemini API
